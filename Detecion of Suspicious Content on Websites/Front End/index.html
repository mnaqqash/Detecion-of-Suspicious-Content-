Detection of Suspicious Content on Websites

To detect suspicious content on websites by crawling the website provided by URL. The working is learning based on data of suspicious websites. Windows firewall blocks or alerts the user about all unauthorized inbound or outbound connection attempts. It allows the user to control which programs can and cannot access the local network or Internet and provides the user with information about an application that makes a connection attemptbut it is not intelligent enough to accommodate user defined context.

In this project, we have designed an intelligent solution based on text mining. The solution is based on algorithms trained on suspicious content. If a user visits such website, our solution will block that content.

Motivation

A lot of websites also called new communication channels are increasingly used by suspicious users who take advantages of this technological feat to publish illegal, suspicious contents and provocation in order to exchange data online and share ideas, anti-state, blasphemous and sectarianism that could affect the security of countries, state, society or institutions. Our project is able to detect such website and block them and prevent the user from such content.
